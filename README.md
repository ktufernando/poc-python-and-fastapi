## About the code

The project uses the following dependencies (libraries), fundamental for the solution:

- [**P**ython](https://www.python.org/): Runtime environment
- [**F**astAPI](https://fastapi.tiangolo.com/): Backend framework
- [**P**eewee](http://docs.peewee-orm.com/en/latest/): ORM for MySql Database
- [**U**vicorn](https://www.uvicorn.org/): Lightning-fast ASGI server implementation
